## ✨ What is this?

**The Universe Engine** is not a normal website.

It is a real-time universe simulation built entirely in the browser using WebGL, Three.js, and custom GLSL shaders. Every particle has mass. Every interaction bends spacetime. Every second, the laws of physics are being calculated and rendered at 60 FPS.

This is what the web looks like from 500 years in the future.

## 🌠 Features

### Universe Background
- 15,000 real-time stars rendered with custom GLSL shaders
- Procedural nebula clouds with additive blending
- Spiral galaxy formations
- Cosmic dust layers
- Slow camera drift through deep space

### ⚫ Black Holes
- 3 black holes with realistic accretion disks
- Relativistic jet beams
- Orbiting debris particles
- Gravitational lensing glow effect
- Event horizon rendering

### 🌀 Gravity Engine
- Mouse cursor acts as a gravitational body
- Click anywhere to create a gravity burst
- Particles orbit, accelerate, and bend trajectories
- Invisible dark matter bodies influence visible particles
- Spacetime fabric grid warps in real time

### 🗺️ Galactic Map
- Interactive 2D map of the universe
- 12 locations: galaxies, black holes, nebulae, wormholes, pulsars
- Click any node to travel there
- Visual travel flash and announcement

### 💥 Reality Fracture
- Randomly triggers every 20–40 seconds
- Screen cracks with branching fracture lines
- Red glow bleeds through the cracks
- Spacetime self-heals with animation
- Manually trigger with `R`

### 🔊 Cosmic Audio Engine
- Deep ambient cosmic hum (layered oscillators)
- All audio generated with Web Audio API — zero files needed
- Audio reacts to every mode change
- Universe collapse alarm sequence
- Time dilation pitch shift

### 🤖 Universe Core AI
- Floating terminal narrates events in real time
- Reacts to every mode activation
- Idle messages about dark matter, quantum states, cosmic events
- Pre-written responses — no API needed

### 💀 Universe Collapse
6-phase destruction sequence:
1. Stellar collapse alarm
2. Galaxy distortion
3. Gravity approaching infinity
4. Black holes merging
5. Singularity formation
6. All matter converging to a point

Then — recreate a new universe from scratch.

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **Three.js** | 3D universe rendering |
| **WebGL / GLSL** | Custom star shaders |
| **GSAP** | Smooth animations and transitions |
| **Vite** | Dev server and build tool |

| **Canvas 2D API** | Spacetime fabric + particles |

universe-engine/

## ⚙️ Run Locally

**Requirements:** Node.js 18+

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/universe-engine.git

# Enter the project
cd universe-engine

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open **http://localhost:5173** in Chrome.
